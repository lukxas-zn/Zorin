¿Qué es Zorin OS?

Haz clic abajo para descubrir el sistema diseñado para ser el puente perfecto.

Empezar Presentación

Hecho en Dublín

Fue creado en Irlanda por los hermanos Artyom y Kyrill Zorin.

Su meta era clara: crear una versión de Linux que fuera tan fácil de usar que cualquier usuario de Windows se sintiera como en casa.

Ver su base

Basado en Gigantes

Zorin OS se construye sobre Ubuntu 22.04 LTS, heredando su estabilidad y seguridad.

¿Cómo se descarga?

Proceso de Descarga

1. Entra a zorin.com

2. Elige Zorin OS Core (Gratis) o Lite.

3. Descarga el archivo ISO y grábalo en un USB con Rufus.

¡Listo para instalar!
